Theme Switcher

This page will teach you how make a theme switcher with JavaScript

FIrst we will need add 2 new classes to our css, those will be the name of the themes, you can add as many new themes as you want, just remember that you will need add a new class for each theme, this time, we will just use 2, a dark mode and a light mode.

.theme-dark {
}

.theme.light {
}


Then, we will need some HTML, we will give the body tag an id, and also a default theme defined by the class name, and that will be the dark theme. also add an checkbox type input, we will give it an id and set it checked because the dark theme is on, also add a onclick event, that will detect when the user clicks the checkbox and change the theme.


And also, some JavaScript, we will save this in a new file called theme-switch.js
function setTheme() {

// Declare the function

let body = document.getElementById("body");

// Declare the body tag with the id we added to the body tag

let checkBox = document.getElementById('theme-switch');

// Declare the checkbox input.


if (checkBox.checked == true) {

// When the checkbox is clicked, it will check if it's checked, if true, will add the dark-theme class, and remove the light-theme class

body.classList.add("theme-dark");
body.classList.remove("theme-light");
} else {

// Else if not, will remove the dark-theme class and add the light theme class

body.classList.add("theme-light");
body.classList.remove("theme-dark");
} }


And we're almost done, then comes the hard part, the css



We will just use a text tag for now, but you can use this with any tag

.theme-dark {
--text-color: white;

/* We use the -- to declare a variable, and we're giving the variable "text-color" the property "white" */

--background-color;
}

.theme-light {
--text-color: black;
--background-color: white;
}

/* Then we will need apply the changes

body {
background-color: var(--background-color);

/* And that's how you apply the variables to the properties.

color: var(--color);
}
